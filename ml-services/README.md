## ML Services

This folder groups **machine learning related services** for FashionCart, including:

- `dss/`: Size & style recommendation (DSS).
- `vton/`: Virtual Try-On model and service.
- `shared/`: Shared utilities, evaluation, and experiments.

Each subfolder may contain:

- `data/`
- `notebooks/`
- `models/`
- `service/`
- `evaluation/`

No implementation code has been added yet.
