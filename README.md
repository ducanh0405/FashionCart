# FashionCart

FashionCart is an e-commerce web platform focused on fashion and clothing for young people. It aims to let users **discover**, **virtually try on**, and **purchase** items from different shops in one place.

### Core Idea

- **Fashion-focused marketplace**: Aggregate products from multiple local brands and stores (starting with HCM City).
- **Virtual Try-On (VTON)**: Integrate an optimized VTON model so users can virtually try outfits before buying.
- **Discovery & social feel**: Over time, evolve toward a social-media-like experience (inspired by platforms such as TikTok) where users can share looks and interact.

### Key Features (Planned)

- **E-commerce essentials**: Product feeds, cart, checkout, user profiles, order history.
- **Communication**: Basic chat/messaging between buyers and shops.
- **Personalization**:
  - Size and style recommendations based on user information.
  - Personalized product feeds.
- **Virtual Try-On (VTON)**:
  - VTON model tailored to fashion use-cases.
  - Acceptable-quality outputs suitable for real-world demos.

### Project Goals (Web App Course)

1. Build an e-commerce platform that includes all necessary core functions.
2. Develop DSS (decision support) and VTON models that work for specific fashion scenarios and produce acceptable results.
3. Deliver an MVP with the main features ready for demo.
4. Prepare presentation slides to clearly present the concept, system, and demo.

> After the course phase, the plan is to run a survey, validate the idea, and if it proves suitable, form a team and continue as a software engineering project targeted for real-world deployment.

### Long-Term Vision (My Project End Goal)

- **Ready to implement**: System design and architecture suitable for real-world scaling and iteration.
- **Infrastructures ready**: Backend, frontend, and ML services wired together with production-ready foundations.
- **Local ecosystem integration**: Connect with local brands and stores in HCM City to onboard real products and partners.
